User Story Card (Story 2)
______________________________________________________________________________
As a 	: developer
I want	: to write a simple Python function and test it with multiple inputs
So that : I can verify correct behavior before integrating it into a larger program

Effort
Level	: 2

Acceptance Criteria
Given	: 	
- I am a developer writing in python

When:
- I run the function

Then: 
- the function prompts me for multiple inputs

When: 
- I test with multiple inputs

Then: 
- The function behaves correctly
And:
- Returns the desired output for the larger program

When: 
- The inputs are invalid

Then: 
- The function exists with a prompt diplaying invalidity
